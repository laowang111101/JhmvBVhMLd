# 图书管理系统（GUI版本） python750

## 项目概述

图书管理系统（GUI版本）是一款使用Python开发的软件，主要用于图书信息的增加、删除、修改、查找以及显示所有图书信息等操作。

## 技术栈

- 开发语言：Python

## 功能模块

- 增加图书信息
- 删除图书信息
- 修改图书信息
- 查找图书信息
- 显示所有图书信息
- 异常处理
- 数据库信息通过配置文件管理

## 系统角色

- 系统管理员：负责图书信息的维护和管理

## 运行环境

- 操作系统：支持主流操作系统（Windows、Linux、macOS）
- Python版本：3.x

## 部署步骤

1. 安装Python环境
2. 安装依赖库
3. 导入数据库文件
4. 运行程序

## 目录结构

```
.
├── main.py              # 程序主入口
├── config.py            # 配置文件，用于管理数据库信息
├── db                   # 数据库文件夹
│   └── books.db         # 数据库文件
└── modules              # 功能模块文件夹
    ├── add_book.py      # 增加图书信息模块
    ├── delete_book.py   # 删除图书信息模块
    ├── modify_book.py   # 修改图书信息模块
    ├── search_book.py   # 查找图书信息模块
    └── show_books.py    # 显示所有图书信息模块
```

## 核心亮点

- 使用图形用户界面（GUI）进行操作，方便用户快速上手
- 数据库信息通过配置文件管理，便于维护
- 异常处理机制，提高系统稳定性
- 支持定制开发，可根据用户需求进行功能拓展与优化
